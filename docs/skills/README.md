# Codex Skills

Skills are task-specific instructions for future Codex and coding-agent work.
They capture project expectations that would otherwise need to be repeated in
every prompt.

Use [`AGENTS.md`](../../AGENTS.md) as the entry point, then read the skill that
matches the task:

- [`frontend-skill.md`](frontend-skill.md): Next.js, React, and workspace UI.
- [`backend-api-skill.md`](backend-api-skill.md): FastAPI routes and schemas.
- [`documentation-skill.md`](documentation-skill.md): GitHub-facing project docs.
- [`i18n-skill.md`](i18n-skill.md): English and Chinese interface preparation.

Skills guide implementation but do not expand task scope. Explicit user
requirements and the project's product boundaries remain authoritative.
